# 🎣 PhishHunter Lite - 스미싱 탐지 웹앱

한국의 최신 스미싱 사례를 탐지하는 웹앱입니다. 사용자가 의심스러운 메시지(문자/카톡/이메일)를 붙여넣으면 AI가 자동으로 분석합니다.

## 🚀 시작하기

### 필수 요구사항
- Node.js 18 이상
- npm 또는 yarn
- OpenAI API 키

### 설치 및 실행

1. **의존성 설치**
```bash
npm install
```

2. **환경 변수 설정**
```bash
cp .env.example .env
```

`.env` 파일을 열고 OpenAI API 키를 입력하세요:
```
VITE_OPENAI_API_KEY=your_openai_api_key_here
```

3. **개발 서버 실행**
```bash
npm run dev
```

4. **브라우저에서 확인**
브라우저에서 `http://localhost:5174` (또는 표시된 주소)로 접속하세요.

## 📋 주요 기능

### 1. 메시지 분석
- 의심스러운 메시지를 텍스트 영역에 붙여넣기
- "분석하기" 버튼 클릭으로 즉시 분석
- 분석 중 로딩 애니메이션 표시

### 2. 분석 결과 표시
- **위험 점수 (3단계)**
  - 🔴 HIGH: 명백한 스미싱/피싱 (점수: 80-100)
  - 🟡 MEDIUM: 의심스러운 요소 있음 (점수: 40-79)
  - 🟢 LOW: 안전해 보임 (점수: 0-39)

- **분석 결과 포함 항목**
  - 위험 점수와 레벨
  - 의심 근거 (최소 3가지 구체적 이유)
  - 대응 가이드 (단계별 행동 지침)
  - 주요 위험 키워드 하이라이트
  - 발견된 URL 및 의심스러운 도메인 표시

### 3. 한국형 스미싱 패턴 감지
다음 최신 사례들을 중점적으로 탐지:
- 카카오/네이버 사칭
- 결제/금융 사기
- 택배/배송 피싱
- 정부/공공기관 사칭
- 고용/리워드 사기

## 🔒 보안 주의사항

⚠️ **중요**: 현재 이 프로젝트는 프론트엔드에서 OpenAI API 키를 직접 사용합니다. 프로덕션 환경에서는 반드시 백엔드 서버를 통해 API를 호출하도록 변경해야 합니다.

- `.env` 파일은 절대 커밋하지 마세요
- API 키가 브라우저에 노출될 수 있습니다
- 프로덕션 배포 시 백엔드 API로 마이그레이션 권장
